# Nine_Image_Compare
Compare one to nine images when use ComfyUI

# Tips
* use 1-9 to select the image you want to compare/可以同时对比1-9张图片

## Usage
1.Installation  
-----

* In the ./ComfyUI/custom_nodes directory, run the following:   

```
git clone https://github.com/smthemex/Nine_Image_Compare.git

```
